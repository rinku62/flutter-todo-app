# Flutter Todo App

## Overview
This is a simple Flutter Todo application that allows users to manage their tasks effectively.

## Features
- Add, edit, and delete tasks.
- Mark tasks as complete.
- User-friendly interface.
- Responsive layout.

## Getting Started
### Prerequisites
- Flutter SDK
- Dart SDK

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/rinku62/flutter-todo-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd flutter-todo-app
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```

### Running the App
To run the app, use the following command:
```bash
flutter run
```
